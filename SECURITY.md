# Security

Holytear Instagram Unfollowers is a static website and browser-console snippet.
There is no backend that receives Instagram account data.

## Verify Before Use

- Review `src/instagram-unfollower.js`.
- Compare the generated snippet hash with the hash shown on `security.html`.
- Review the VirusTotal report linked on the security page.
- Review the Semgrep workflow:
  <https://github.com/Holytear/instagram-unfollowers/actions/workflows/semgrep.yml>

## Current Snippet Hash

See `security.html` or run `npm run build` and hash `dist/instagram-unfollower.one-line.js`.
